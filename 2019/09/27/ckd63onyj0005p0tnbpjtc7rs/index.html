js位操作符

发表于 2019-09-27 | 分类于 js | 评论数：

js有个很少用的操作符叫做 位操作符

其中包括：与& 或| 异或^ 非~ 右移>>左移 <<

具体操作符的运算法则自行搜索（eg: ~为二进制反码,等于操作值的负值减1）

都只针对整数（JS数值保存都以IEEE-754 64位格式存储，而位操作符会先将64位转成32位进行操作再转回64位）32位：第一位为符号位 0为正 1为负数

这里说些比较有可能用上的用法：

  1. 使用~~，>>,<<,|来取整

    ~~3.14 = 3

    ~3.14+1=3

    3.14 >> 0 = 3 //位移0位 就等于只是转换

    3.14 << 0 = 3 3.14 | 0 = 3 3.14 >>> 0 = 3(>>>不可对负数取整)

    3.14|0=3

    注意：~~-3.14 = -3 其它的一样

  2. 使用&运算符判断一个数的奇偶

    偶数 & 1 = 0

    奇数 & 1 = 1

    那么0&1=0,1&1=1

  3. 使用^来完成值交换 (原理是 X^Y^Y=X)

    a = 1

    b = 2

    a ^= b // a=1^2

    b ^= a // b=1^2^2 =1

    a ^= b // a= 1^2^1 =2

    结果a=2，b=1

  4. 使用&,>>,|来完成rgb值和16进制颜色值之间的转换 (过程太麻烦就不深挖了)

    运行RGBToHex(“rgb(255,255,255)”)返回”#ffffff”
